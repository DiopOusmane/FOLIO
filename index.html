QUI SUIS-JE ?

Je m'appelle Ousmane, développeur web et graphiste passionné par le design et la création numérique. Je combine créativité, rigueur et sens du détail pour concevoir des projets alliant esthétique et fonctionnalité. Curieux et motivé, j'aime apprendre continuellement et relever de nouveaux défis. Je privilégie le travail en équipe et la collaboration, convaincu que les meilleures idées naissent souvent d'échanges constructifs. Mon objectif est de contribuer à des projets innovants qui ont un impact positif et durable.

InfoCountry

InfoCountry est une application web interactive permettant d'obtenir rapidement des informations détaillées sur tous les pays du monde. Ce projet a vu le jour dans le cadre d'un portfolio personnel pour mettre en avant mes compétences.

Fonctionnalités principales :

  • Recherche d'un pays par son nom
  • Affichage du drapeau, de la population, de la région, de la capitale, etc.
  • Traduction automatique des régions en français
  • Interface responsive et conviviale
  • Gestion des erreurs (pays non trouvé, etc.)

Points forts ou originalités :

  • Traduction dynamique des régions pour une meilleure expérience utilisateur francophone
  • Utilisation d'une API externe pour garantir des données à jour
  • Interface épurée et accessible sur tous supports
  • Code optimisé et structuré (utilisation de switch case, gestion des erreurs)
HTML CSS JavaScript API REST Countries
